# 布局

# 使用

由 goLayout 和 goLayoutCol 组成，goLayout为主容器，goLayoutCol为列容器，一个goLayoutCol表示一列

  <template>
    <goLayout @onResizeEnd="onResizeEnd">
      <goLayoutCol :width="300" unit="px">left content</goLayoutCol>
      <goLayoutCol :width="400" unit="px">middle content</goLayoutCol>
      <goLayoutCol :width="20" unit="%">right content</goLayoutCol>
    </goLayout>
  </template>

# goLayoutCol属性说明

属性 说明 类型 默认值
width 宽度 Number 33.3
unit 宽度单位( percent 、 px ) String percent
minWidth 最小宽度 Number 10
minWidthUnit 最小宽度单位( percent 、 px ) String percent
leftResizeWidth 左触发区域宽度( px ) Number 4
rightResizeWidth 右触发区域宽度 ( px ) Number 4

# goLayout事件说明

事件名称 说明 参数 说明
onResizeEnd 用户调整宽度结束 { widthList } 调整后各列宽度和单位
