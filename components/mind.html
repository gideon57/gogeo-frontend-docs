# 脑图

baseApp封装了 jsmind (opens new window) 提供给appkit使用

使用 goMind 组件，并传入 宽 , 高 , option, 即可渲染 示例如下

# 使用

  <template>
  <div class="">
    <goMind width="300px" height="200px" :option="mindOption" />
  </div>
</template>
<script>
export default {
  data() {
    return {
      mindOption: [
        { id: 'root', isroot: true, topic: 'jsMind' },
        { id: 'sub1', parentid: 'root', topic: 'sub1' },
        { id: 'sub2', parentid: 'root', topic: 'sub2' },
      ]
    }
  }
}
</script>

# 节点样式配置

export default {

  const nodeData = [id: 'sub2-1', parentid: 'sub2', topic: 'sub2-1']
  const nodeStyle = ['leading-line-color': 'red', 'foreground-color': 'blue']

  data() {
    return {
      mindOption: [
        { ...nodeData, ...nodeStyle }
        # { id: 'sub2-1', parentid: 'sub2', topic: 'sub2-1', 'leading-line-color': 'red', 'foreground-color': 'blue' }
      ]
    }
  }
}

# 样式配置属性说明

属性 说明 类型 备注
background-color 节点的背景颜色 string #1abc9c, blue
foreground-color 节点的文本颜色 string #1abc9c, blue
width 节点的宽度(px) number 400
height 节点的高度(px) number 400
font-size 节点的文本字体大小(px) number 32
font-weight 节点的文本重量 string bold
font-style 节点的文本样式 string italic
background-image 节点的背景图片 url/base64
background-rotation 节点的背景图片旋转角度 number 30
leading-line-color 节点的引导线颜色 number #1abc9c, blue
