# 属性说明

属性 说明 类型 默认值
dashboardPortalId 查询配置参数 string
dashboardId 查询配置参数 string
filterKeys 查询配置参数 array
customFiltersConfig 外部传入参数，传入时不再调接口 array
popover 是否使用popover弹窗 boolean true
width popover宽度 number 600
trigerIcon popover按钮icon string icon-a-shaixuanqi11
trigerText popover按钮文字 string 筛选
submitBtnShow 是否显示确认按钮 boolean true
resetBtnShow 是否显示重置按钮 boolean true
submitOnReset 重置时是否触发submit事件 boolean false
rows 一行几个表单项 number 2
labelWidth 表单label宽度 number 100
labelPosition 表单label位置 string left

# 事件说明

属性 说明 参数 说明
onReset 用户点击重置触发 { widgetList }
onSearch 用户点击查询触发 { widgetList }
onItemChange 表单项改变触发 {i,filter}
