# 挂件

使用 goWidget 组件，并传入配置信息 即可渲染 示例如下

# 使用

  <template>
        <go-widget v-model="widgetInfo"></go-widget>
  </div>
</template>
<script>
export default {
  data() {
    return {
      widgetInfo: [
        bg:'red',
        list:[
          {
            url:'',
            title:'',
            queryParams:null,
            bg:'green'
          }
        ]
      ]
    }
  }
}
</script>

# 配置信息属性说明

属性 说明 类型 是否必须 备注
list 挂件列表 Array 是 详见list属性说明
bg 挂件列表容器背景色 string 否 可不传，默认#F5F7FA

# list属性说明

属性 说明 类型 是否必须 备注
url 发布挂件地址 string 是 发布挂件
secondTitle 挂件标题 string 否 没有则不显示
queryParams 需要追加的筛选条件 object 否 筛选条件
bg 每个挂件的默认背景色 string 否 可不传，默认#FFF

# 更新挂件展示内容

  • 修改对应 widgetInfo.list 中对应挂件对象的 queryParams 即可

  • 挂件title同上
