# 按钮权限

baseApp提供 $btnPermission 给appkit使用

示例如下

<template>
  <go-button v-if="$btnPermission('permission_key')">权限按钮</go-button>
</template>

# 下载文件

baseApp提供 $exportBolb 给appkit使用

需要两个参数

  • fileId 为文件ID
  • fileName 为文件名称

示例如下


methods:{
  exportBolb() {
    let fileId = '268500'
    let fileName = 'demo.xlsx'
    this.$exportBolb(fileId, fileName)
  }
}

# 正则

# dayjs

已将 dayjs (opens new window) 挂载在baseApp实例原型上，可按示例对日期/时间进行转换

  let time = new Date().getTime()
  let day = this.$dayjs(time).format('YYYY年MM月DD日')
  console.log('dataFormat', day)
  #2024年05月22日

# lodash

baseApp封装 lodash (opens new window) 的 cloneDeep 给appkit直接使用

  let cloneData = this.$cloneDeep(data)

其他方法可直接在appkit中按需引入使用，示例如下

  import {debounce} from 'lodash'
  debounceFun: debounce(function () {
    # do something
  }, 1000),

# turf

baseApp封装 turfjs (opens new window) 的一些常用API提供给appkit使用

其中包括：point , distance , lineString , length , polygon , area , points , center , kinks

使用方式如下：

  this.$turf.point(options)

其他方法可直接在appkit中按需引入使用，示例如下

  import {point as turf_point} from '@turf/turf'
  turf_point(options)
