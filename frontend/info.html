# 系统信息

# 介绍

子应用可能需要获取当前登录人的个人、组织、配置等信息

# 个人信息

在computed中引入定义需要的信息

  computed: {
    #token
    token(){
      return this.$store.state.user.token
    },
    #用户名
    name(){
      return this.$store.state.user.name
    }
  }

# 组织信息

在computed中引入定义需要的信息

  computed: {
    #租户编码
    tenantCode() {
      return this.$store.state.user.tenantCode
    },
    #租户名称
    tenantName() {
      return this.$store.state.user.tenantName
    }
  }

# 配置信息

在computed中引入定义需要的信息

  computed: {
    #主题色
     goGeoPrimary() {
      return this.$store.state.settings.goGeoPrimary
    },
    #系统logo
    goGeoLogo() {
      return this.$store.state.settings.goGeoLogo
    },
    #系统全称
    goGeoProductName() {
      return this.$store.state.settings.goGeoProductName
    },
    #系统简称
    goGeoProductShortName() {
      return this.$store.state.settings.goGeoProductShortName
    },
  }

在computed中引入定义需要的信息

    }
  }

# 子应用信息

在子应用获取当前页面所属的 应用 和 appkit 信息

baseApp会将子应用相关信息放在 $route/meta 中

建议使用依赖注入 provide / inject

示例如下

# 父组件 provide

  export default {

    # computed中定义变量
    computed: {
      appCode() {
        return this.$route.meta.appInfo.id
      },
      appPageCode() {
        return this.$route.meta.appPageInfo.id
      }
    },


    # provide 提供数据
    provide() {
      return {
        appCode: this.appCode,
        appPageCode:this.appPageCode,
      }
    }
  }

# 子组件 inject

  export default {

    #inject 注入数据
    inject: ['appCode','appPageCode'],

    #使用数据
    created() {
      console.log(this.appCode)
      # app_gg_xxxxxx
    },
  }

# meta说明

属性 说明 备注
isApp 是否app
icon app/菜单图标
title app/菜单名称
appInfo 见appInfo说明
appPageInfo 见appPageInfo说明

# appInfo说明

属性 说明 备注
name app名称
id appCode

# appPageInfo说明

属性 说明 备注
id appCode
parentId 父路由id 即应用id
appkit 该页面所属appkit的名称 appkit/package.json的包名
