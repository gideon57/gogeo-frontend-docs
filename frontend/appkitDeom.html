# 左右布局

appkit/subdivisionPlan/index.vue

一个go-row会把整行分成24份

每个go-col的 span代表该col占据24中的多少份

如：左边crud占据 16/24宽度 右边挂件 8/24宽度

   <go-row class="layoutRow">
     <go-col :span="16" class="layoutCol">
     </go-col>
      <go-col :span="24" class="layoutCol">
     </go-col>
   </go-row>

# 左侧布局css

appkit/subdivisionPlan/index.vue

本页面curd部分由 筛选条件 + 表格 + 分页 组成

定义 分页和 筛选条件 高度分别为 40 和 70

  --pageHeight:40px;
  --filterFormHeight:70px;

则列表高度为 总高度减去其他

.tableWra {
  height: calc(100% - var(--filterFormHeight) - var(--pageHeight));
}

再设置go-table高度为100% 可铺满全屏并流式滚动

<go-table height="100%" >

    myTitle="编辑方案"

    append-to-body
    center
    width="500px"
    @close="editPlanObj.visible = null"
