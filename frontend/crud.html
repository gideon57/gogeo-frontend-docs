# 初始配置和概念

  • 引入并使用tableMixin（搜索并复制）

    会自动生成tableParams查询参数，包括当前页和页数

      import { tableMixin } from '@/mixins/table'
  mixins: [tableMixin],

  • 刷新查询必须使用 onRefresh 方法名

  • 分页查询必须使用 getList 方法名
